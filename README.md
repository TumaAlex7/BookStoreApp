## Описание

Учебный проект по объектно-ориентированному программированию на C#. Приложение представляет собой систему управления книжным магазином с возможностью аренды книг.

### Основные компоненты

- **`Book`** — класс книги с инкапсулированными данными (ISBN, название, автор, цена, остаток)
- **`Catalog`** — коллекция книг с индексаторами для различных способов доступа
- **`BookStore`** — фасад для работы с каталогом (аренда, возврат, изменение цены)

### Реализованные концепции ООП

- ✅ Инкапсуляция с валидацией данных
- ✅ Свойства (properties) с get/set
- ✅ Конструкторы с параметрами
- ✅ Индексаторы (по ISBN, по индексу, по автору и номеру)
- ✅ Реализация `IReadOnlyCollection<Book>`
- ✅ Собственный `IEnumerator`
- ✅ Обработка исключений

## О RuntimeIdentifier

В проекте явно указан `RuntimeIdentifier` для корректной работы на Arch Linux:

```xml
<RuntimeIdentifier>linux-x64</RuntimeIdentifier>
```

Если ваша ОС не Linux, рекомендуется удалить эту строку для автоматического определения Runtime Identifier


### Зачем это нужно?

- На моём Arch Linux .NET неправильно определяет Runtime Identifier
- Без явного указания возникает ошибка: `Unable to find package Microsoft.NETCore.App.Host.arch-x64`
- Явное указание `linux-x64` решает эту проблему
